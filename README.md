🦷 Dental Clinic Admin Panel – ASP.NET Core 8 MVC

A modern, lightweight Dental Clinic Management System built using ASP.NET Core 8 MVC.
Designed for real-world clinic workflows with a clean admin dashboard, appointment handling, prescriptions, and professional invoicing. </br>

🚀 Features
</br>
</br>
📊 Dashboard

- Today’s revenue
- Monthly & yearly revenue
- Pending invoices
- Today’s appointments
- Upcoming appointments table
- Clean admin-style UI (Bootstrap)

📅 Appointment Management
- Create, edit & delete appointments
- Branch selection (Islamabad, Peshawar, Karachi)
- Default status handling (Pending, Completed, Cancelled)
- One-click status updates
- Pagination for large datasets

📝 Prescription Management

- Create prescriptions linked to appointments
- Multiple medicines per prescription
- One-click Save & Print
- Clean printable prescription layout

🧾 Invoice Management

- Professional invoice generation
- Auto invoice number
- Subtotal, discount, paid & remaining amount
- Partial payment support
- Edit & reprint invoices
- Paginated invoice list

🔔 User Experience

- SweetAlert confirmations
- Toast notifications (Create / Edit / Delete)
- Clean Bootstrap admin layout
- Responsive & clinic-friendly UI

🛠️ Tech Stack

- ASP.NET Core 8 MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- JavaScript (Fetch API)

SweetAlert2

Toastr Notifications
